# Matlab-Scripts-For-Landsat-Kelp
This repository contains a series of matlab script for introductory to data analysis using a spatial data.The data used here is provided as a NetCDF file, which stores kelp canopy biomass and area estimates on a consistent gridded spatial framework along with supporting metadata. The data can be readily visualized at https://kelpwatch.org/ and downloaded here: https://portal.edirepository.org/nis/mapbrowse?scope=knb-lter-sbc&identifier=74
